概要
自主制作｜ポートフォリオ
ツール/言語
Visual Studio Code / HTML / CSS / jQuery / Illustrator / Photoshop / Canva
制作時間
2ヶ月
・ワイヤーフレーム…7日
・デザインカンプ…8日
・素材収集・制作…2週間

これまでの経験や制作物をわかりやすく伝えるために、シンプルで見やすい構成を意識しながらポートフォリオサイトを制作しました。

このサイトを通して、私の雰囲気や大切にしている考え方などが少しでも伝われば幸いです。

  • Retouch
概要
訓練校課題｜アプリ紹介バナー
ツール
Illustrator / Photoshop
制作時間
4時間

1日5分で始められるマインドフルネスアプリ「Quietly」の紹介用バナーを制作しました。
「忙しいあなたのための、5分リセット」をコンセプトに、仕事中や通勤の合間に目にした際、“時間がかからず、無理なく続けられそう” と直感的に伝わり、心理的なハードルを下げる構成を意識しています。

アプリ画面を大きく配置することで、実際の利用シーンを想像しやすくし、キャッチコピーとビジュアルの両面からサービスの魅力が伝わるバナーを目指しました。

概要
自主制作｜PC壁紙
ツール
Photoshop
制作時間
2時間

パズルのピースがはめ込まれ、一部が抜け落ちた瞬間を切り取ったような、遊び心のあるPC用壁紙を制作しました。
ハロウィンらしい暖色を基調にしながらも、光のにじみ具合やコントラストを調整し、背景として使った際に主張しすぎないバランスを意識しています。
くり抜かれたピース部分と、動かしたピースには、それぞれ影の強さや方向を変えることで、平面的になりがちな壁紙に「触れそうな質感」や「動きを連想させる要素」を取り入れました。
画面を開いた瞬間に、奥行きや空間の広がりを感じられるよう意識しています。
視線を集めるモチーフでありながら、作業の邪魔にならないよう彩度や明度を調整し、背景として心地よく使える壁紙デザインに仕上げています。

  • Retouch
概要
訓練校課題｜タイポグラフィ
ツール
Illustrator
授業日数
1日

どこかなつかしく、やさしい雰囲気を感じられるリソグラフ印刷風のタイポグラフィ作品です。
Illustratorで長方形や円、半円といったシンプルな図形を組み合わせ、図形をパズルのように配置しながらアルファベットを形づくりました。
乗算効果による色の重なりや、インクがかすれたような質感を加えることで、手刷り印刷ならではの温もりや素朴な風合いを表現しています。
レトロでかわいらしい配色と、少し不完全なテクスチャーをあえて残すことで、見ていて楽しく、親しみやすいタイポグラフィを目指しました。

Sacred Oxen

  • Design
  • Retouch
概要
訓練校課題｜壁に描かれた神話
ツール
Photoshop
授業日数
1日

訓練校の授業課題として、角度のついた壁面写真にグラフィックを合成するフォトコンポジット作品を制作しました。
架空の展示イベントを想定し、神話的モチーフを用いた世界観のあるビジュアルに仕上げています。
Photoshopの消点フィルターを使用し、壁面のパースに合わせてグラフィックを変形・配置することで、空間に自然に溶け込むような一体感を目指しました。
合成時には、壁の傾きに沿った形状調整、光源を意識した明暗と影のなじませ、コンクリート壁の質感に合わせたトーン調整などを行い、平面的に見えないように工夫しています。
写真とグラフィックの境界を感じさせないことを意識し、その場に描かれているように見せる表現を追求しました。

概要
自主制作｜デジタルアート
ツール
Photoshop
制作時間
1.5時間

神秘的な光に包まれた森の広場で、静かに佇む鹿を描いたデジタルアート作品です。
森に迷い込んだときに出会った、ささやかな友をテーマに、静かな安心感と胸に残る不思議な余韻が同時に感じられる世界観を目指しました。
Adobe Fireflyで生成したイメージをベースに、油彩風のタッチやキャンバスのテクスチャを重ねながら、絵画のような温もりと奥行きを演出しています。
ノートの表紙や部屋に飾るアートとして、日常の中でふと視線を向けたとき、長く眺めていたい一枚に仕上げました。

Flying Sheep

  • Design
  • Retouch
概要
自主制作｜コラージュ作品
ツール
Illustrator / Photoshop
制作時間
1.5時間

風船に身をゆだねながら、上空を旅する迷える子羊を描いたコラージュ作品です。
自分の意思ではなく、風船の流れに身を委ねながら進む姿に、行き先のわからない不安や人生の曖昧さ、その先に待っているかもしれない希望の兆しを重ね合わせました。
やわらかなパステルカラーと虹のグラデーションによって、現実と空想のあいだを漂っているような、幻想的で温かみのある世界観を表現しています。
タイトルは風船をモチーフに立体的にデザインし、ビジュアル全体と自然に調和するよう質感や影の強さを調整しました。
見る人が物語を想像しながら、静かな余韻を感じられる作品を目指しています。
