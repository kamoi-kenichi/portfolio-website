  • Direction
  • Design
  • Copywriting
  • Coding
  • Retouch
概要
自主制作｜ポートフォリオ
ツール/言語
Visual Studio Code / HTML / CSS / jQuery / Illustrator / Photoshop / Canva
制作時間
2ヶ月
・ワイヤーフレーム…7日
・デザインカンプ…8日
・素材収集・制作…2週間
・コーディング…4週間

これまでの経験や制作物をわかりやすく伝えるために、シンプルで見やすい構成を意識しながらポートフォリオサイトを制作しました。
配色はライトベージュとチャコールグレーを基調に、アクセントカラーにモーブグレーを使用し、やわらかい雰囲気で落ち着いた配色の北欧ナチュラル感を演出しました。
細かい要素を足すことよりも、ミニマルな配色やシンプルで直感的な導線にこだわり、閲覧者の負担を減らしつつ、必要な情報に自然と辿り着ける構成にしています。
全体を通して、派手さよりも安心して見てもらえることを大切にしながら、作品としての完成度と自分らしさの両立を目指しました。
このサイトを通して、私の雰囲気や大切にしている考え方などが少しでも伝われば幸いです。

Visual Studio Code / HTML / CSS / jQuery / PHP / MAMP / Illustrator / Photoshop
制作時間
2日

自分の名前を入力して、文字コードをもとに「スピリットアニマル」を判定する診断アプリを制作しました。
名前を利用したシンプルなアルゴリズム設計で、ランダムではなく「名前に基づいた」診断結果を返す仕組みを実装しています。
ローディング画面やタイトルのエフェクト、グラデーションボタン等のアニメーションで「占いコンテンツらしいワクワク感」を演出し、印象に残るUI/UXを意識しながら制作しました。
訓練校で学んだPHPのフォーム処理や文字列操作、アニメーションを使った演出等の習熟度を図る目的で取り組みました。
今後は、複数の質問に答えるタイプの診断や、SNSシェア用のOGP画像生成などにもチャレンジしていきたいと考えています。

  • Direction
  • Design

概要
自主制作｜架空の歯科医院サイト
ツール/言語
Visual Studio Code / HTML / CSS / jQuery / Illustrator / Photoshop / Canva
制作時間
2週間
・ワイヤーフレーム…1日
・デザインカンプ…3日
・素材収集・制作…3日
・コーディング…1週間

コンセプト設計からデザイン・実装まで、一連の制作フローを体験する訓練の一環として、架空の歯科医院サイトを制作しました。
“通いやすさ” と “相談しやすさ” を軸に、ユーザーが不安を感じやすい歯科診療のイメージを少しでも和らげることを目標としています。

また、丸みのある輪郭や円形の写真・装飾を取り入れることで、歯科医院特有の“緊張感”を軽減。
ひとりひとりの生活に寄り添ったケアを大切にする歯科医院のイメージを、全体のトーンやふんわりとしたアニメーションで表現しました。

  • Design
  • Retouch
概要
自主制作｜架空の北欧旅行バナー
ツール
Illustrator / Photoshop
制作時間
4時間

デンマーク・ニューハウンの観光プロモーションバナーを制作しました。
淡いブルー系の配色とミニマルなレイアウトで統一し、港町に流れる爽やかな海風と、北欧らしい穏やかな空気感を表現しています。

  • Design
  • Retouch
概要
自主制作｜冬物ファッションセール
ツール
Illustrator
制作時間
4時間

架空のアパレルECサイト『Intp’s shop』の冬物セールバナーを制作しました。
ターゲット層は10〜20代の女性を想定し、パステルカラーの空にネオンがふわりと灯るような"韓国風ネオンのかわいらしさ"と"北欧の静けさや透明感"を掛け合わせ、軽やかで透明感のある冬の雰囲気を感じさせるような優しいトーンや配色でデザインしています。

広告バナー

マインドフルネスアプリ

  • Design
  • Retouch
概要
訓練校課題｜アプリ紹介バナー
ツール
Illustrator / Photoshop
制作時間
4時間

1日5分で始められるマインドフルネスアプリ「Quietly」の紹介用バナーを制作しました。
「忙しいあなたのための、5分リセット」をコンセプトに、仕事中や通勤の合間に目にした際、“時間がかからず、無理なく続けられそう” と直感的に伝わり、心理的なハードルを下げる構成を意識しています。
ストレスを感じている20代後半〜40代のビジネスパーソンをターゲットに、ネイビーを基調とした落ち着きのある配色に、パステルグリーンをアクセントとして用いることで、安心感の中に「小さな変化」や「始めやすさ」を表現しました。
背景には左下から右上へ流れるグラデーションと、波打つようなシェイプを配置し、「呼吸」「時間の流れ」「静かな朝」を視覚的に表現しています。
アプリ画面を大きく配置することで、実際の利用シーンを想像しやすくし、キャッチコピーとビジュアルの両面からサービスの魅力が伝わるバナーを目指しました。

合成作品

チェキ風PC壁紙

  • Design
  • Retouch
概要
自主制作｜PC壁紙
ツール
Photoshop
制作時間
3時間

チェキのような温かみのある質感を表現した、PC用オリジナル壁紙を制作しました。
写真全体にスクラッチや粒状感を加えることで、あえてダメージを残したレトロな雰囲気に仕上げています。
彩度を抑えたやさしい色調と、少し色あせたトーンによって、なつかしさと落ち着いた空気感を演出しました。
ポラロイド写真を壁に飾ったような感覚で、日常の作業画面にそっと馴染むデザインを意識しています。
写真をただ並べるのではなく、質感・色味・余白のバランスを整えることで、インテリアの一部のように使えるPC壁紙を目指した作品です。

合成作品

パズル風PC壁紙

  • Design
  • Retouch
概要
自主制作｜PC壁紙
ツール
Photoshop
制作時間
2時間

パズルのピースがはめ込まれ、一部が抜け落ちた瞬間を切り取ったような、遊び心のあるPC用壁紙を制作しました。
ハロウィンらしい暖色を基調にしながらも、光のにじみ具合やコントラストを調整し、背景として使った際に主張しすぎないバランスを意識しています。
くり抜かれたピース部分と、動かしたピースには、それぞれ影の強さや方向を変えることで、平面的になりがちな壁紙に「触れそうな質感」や「動きを連想させる要素」を取り入れました。
画面を開いた瞬間に、奥行きや空間の広がりを感じられるよう意識しています。
視線を集めるモチーフでありながら、作業の邪魔にならないよう彩度や明度を調整し、背景として心地よく使える壁紙デザインに仕上げています。

リソグラフ印刷風文字

  • Design
  • Retouch
概要
訓練校課題｜タイポグラフィ
ツール
Illustrator
授業日数
1日

どこかなつかしく、やさしい雰囲気を感じられるリソグラフ印刷風のタイポグラフィ作品です。
Illustratorで長方形や円、半円といったシンプルな図形を組み合わせ、図形をパズルのように配置しながらアルファベットを形づくりました。
乗算効果による色の重なりや、インクがかすれたような質感を加えることで、手刷り印刷ならではの温もりや素朴な風合いを表現しています。
レトロでかわいらしい配色と、少し不完全なテクスチャーをあえて残すことで、見ていて楽しく、親しみやすいタイポグラフィを目指しました。
