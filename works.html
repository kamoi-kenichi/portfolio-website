Lumapuyo

JavaScriptで開発した、スマホ操作に対応したオリジナル落ちものパズルゲームです。 スワイプ操作による直感的なプレイと、連鎖による爽快感を重視し、 繰り返し遊びたくなる体験を目指して設計・実装しました。

  • Game Logic
  • JavaScript
  • Touch UI
  • Animation
  • Sound
概要
スマホ操作対応のオリジナル落ちものパズルゲーム
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript
制作期間
約2週間
（設計3日 / デザイン2日 / 素材2日 / 実装7日）

工夫したポイント

盤面管理や当たり判定、連鎖処理などのゲームロジックをJavaScriptで実装し、 状態管理を意識した設計を行いました。

また、スマホ操作でも快適に遊べるよう、タッチ操作の最適化にも工夫しています。

主な技術ポイント：
・スマホ操作を前提にしたスワイプ / タップ入力設計
・連鎖数に応じて変化する演出・効果音・スコア表示
・状態管理を分離したゲームループ構造
・requestAnimationFrameベースの描画制御
・UIレイヤー分離による演出制御設計

素材はすべてライセンス確認済み、または自作素材を使用しています。

企画・操作設計・演出・UI・音響まで一貫して設計・実装しました。

描画・演出・入力処理を分離し、保守性と拡張性を重視して設計しています。

App

LIMITSルーレット

JavaScriptで開発した、デジタルアートバトル「LIMITS」の個人練習向けテーマルーレット＋20分タイマーアプリです。

  • App
  • JavaScript
  • Animation
概要
具体（モノ）と抽象（テーマ）を組み合わせて出題し、20分タイマーを自動開始する練習アプリ
担当
企画 / UIデザイン / コーディング / アニメーション実装
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript
制作期間
約2日
（設計1日 / 実装1日）

課題と解決

練習時に課題になりやすい「テーマ決めに迷う時間」を減らし、発想から制作開始までをスムーズにできるよう設計しています。 また、テーマ確定から20分タイマー開始までを1画面で完結させることで、 本番に近い緊張感と高揚感を再現できる構成を目指しました。

具体（モノ）と抽象（テーマ）をルーレットで組み合わせ、両方の結果が確定すると、 公式ルールと同じ20分タイマーが自動でスタートする仕様にしています。

デザイン面では「北欧ナチュラル × 犬」をコンセプトに、集中を妨げない配色と余白設計を意識しました。

さらに、START時とTIME UP時には犬が跳び上がってぴょこんとおじぎするアニメーションと効果音を加え、 練習の緊張感の中にも親しみやすさを感じられる演出にしています。

因果の刃

  • Graphic Design
  • Visual Design
概要
その一言、誰を傷つけてる。
ツール / 言語
Photoshop
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

ワープ・ゆがみツールを使用し、言葉を刃物状に再構成。
刺さった瞬間を起点にひび割れを広げることで、誹謗中傷が内面の崩壊へと繋がる因果関係を表現しました。
また、グリッチ・ノイズによって不安定な心理状態を演出し、視線誘導（左→右）で因果の流れを設計しています。

深夜の閉ざされた空間を舞台に、孤独と内面の歪みを表現。

攻撃的な誹謗中傷は一時的な快感を伴う一方で、 自分自身の時間や精神を消耗させる行為でもあります。
本作品ではその構造を「因果」として捉え、鑑賞者が自身の言動を見つめ直すきっかけとなることを意図しました。

Design

冬物セールバナー

架空のアパレルECサイト『Intp’s shop』の冬物セールバナーを制作しました。

  • Graphic Design
  • Visual Design
概要
ファッション広告バナー
ツール / 言語
Illustrator
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

ターゲット層は10〜20代女性を想定し、「手に取りたくなる軽やかさ」と「冬らしい透明感」を両立することを意識しました。
パステル調の空にネオンがふわりと灯るようなビジュアルで、韓国風のトレンド感と北欧の静けさを掛け合わせ、やさしく目に留まるデザインに仕上げています。

背景にはラベンダーとピンクのグラデーションを採用し、冬の澄んだ空気感とやわらかな温もりを表現しました。

ネオン文字は強すぎない発光に調整することで、視認性を確保しながらも、全体のやさしいトーンを崩さないよう設計しています。

セール情報は視線の流れを意識して配置し、短時間でも内容が把握できるように構成しました。

トレンド感と訴求力のバランスを取りながら、SNS上でも目に留まりやすいビジュアルを目指しています。

視認性・訴求力・世界観の3点のバランスを意識し、「かわいい」だけで終わらない実用性のあるバナーを目指しました。

ちいさな番人

  • Graphic Design
  • Visual Design
概要
映画キービジュアル
ツール / 言語
Photoshop
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

稲妻や満月、ゴーストといったハロウィンモチーフを取り入れながら、ダークな世界観の中にやわらかさを残すことで、子どもから大人まで楽しめるハロウィンファンタジーを目指しました。

キャラクターを画面中央に配置し、周囲の光やモチーフが自然に視線を集める構図にすることで、物語の主軸が直感的に伝わるよう設計しています。

タイトルロゴは扉のアーチラインに沿わせることで、物語の入口を象徴し、ビジュアル全体と一体化するよう意識しました。

ハロウィンの夜にだけ開く、不思議な扉。
その前に佇むのは、かぼちゃの上に乗っている小さな番人。

かわいらしさの中に使命感を感じさせるキャラクターを中心に、ダークファンタジーと童話的要素を融合した世界観を描いています。

夜の静けさと光のコントラストを意識することで、不安と安心が同時に存在するような空気感を演出しました。

夜の片隅でそっと世界を支える、小さな番人の静かな冒険を描いたキービジュアルです。

キャラクターと背景が一体となって物語を伝える構成にすることで、一枚のビジュアルで世界観が伝わるよう意識しています。

猫耳ヘッドフォン

猫耳デザインのヘッドフォン「Meow Tunes」の広告バナーです。

  • Visual Design
概要
ヘッドフォン広告バナー
ツール / 言語
Photoshop / Illustrator
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

「ねこみみ × Music」というコンセプトを軸に、ターゲットである10〜20代の若年層に刺さるよう、ポップで親しみやすい配色と柔らかいトーンで設計しました。

商品であるヘッドフォンに自然と視線が集まるよう、中央配置とコントラストを意識し、視認性と訴求力のバランスを調整しています。

キャッチコピーは余白を活かして配置することで、情報の詰め込みすぎを避け、直感的に魅力が伝わる構成にしています。

猫好きで流行に敏感な10〜20代の若者をターゲットに、猫耳デザインのヘッドフォン「Meow Tunes」の広告バナーを制作しました。

かわいらしさだけでなく、「日常に音楽を取り入れる楽しさ」が伝わるよう、ポップなビジュアルと軽やかな世界観で構成しています。

視線誘導と情報整理を意識することで、一目で商品の魅力が伝わる広告表現を目指しました。

Nocturne de Vigne

高級フレンチレストランで提供されるクラフトワイン向けの限定ヴィンテージを想定した広告バナーです。

  • Graphic Design
  • Visual Design
概要
ワイン広告バナー
ツール / 言語
Photoshop / Illustrator
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

ぶどう畑の夜想曲をコンセプトに、収穫直前の葡萄が持つ艶やかさと、ワインになる前の一瞬の美しさをテーマに設計しました。

果粒の透明感や水滴の質感を強調することで、高糖度・完熟期ならではの瑞々しさと上質感を表現しています。

主役である葡萄に自然と視線が集まるよう、暗めの背景とコントラストをつけることで、高級感と視認性を両立させました。

背景には深みのあるパープルを使用し、夜の静けさと気品を感じさせる空気感を演出しています。

高級フレンチレストランで提供されるクラフトワイン向け限定ヴィンテージを想定した広告バナーです。

ぶどうを“ワインになる前の芸術品”として捉え、素材そのものの価値を引き立てるビジュアル設計を行いました。

タイトルにはゴールドのタイポグラフィを採用し、視線を誘導しながら、限定性と上質感を強調しています。

視覚的な質感表現によって、味や香りまで想起させることを意識しました。

高級感・特別感・静けさを一枚のビジュアルで伝えることを意識した作品です。

TIME IS MONEY

時間とお金、働き方をテーマにした転職・キャリア広告です。

  • Graphic Design
  • Visual Design
概要
転職・キャリア広告
ツール / 言語
Photoshop / Illustrator
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

コインの上で溶けるろうそくをモチーフに、消費され続ける時間と、気づかぬうちに削られていく人生を視覚的に表現しました。

暗い背景と光のコントラストを活かすことで、ろうそくの存在感を強調し、自然と視線が集まる構図にしています。

転職広告として過度に煽るのではなく、静かに問いかけるトーンにすることで、「自分ごと」として考えさせる余白を持たせました。

視覚的な比喩表現によって、言葉に頼らずにメッセージが伝わる設計を意識しました。

時間とお金、そして働き方の関係に焦点を当てた転職・キャリア広告です。

「TIME IS MONEY」という言葉を単なる格言ではなく、人生の選択を考えるきっかけとして再解釈し、視覚的に訴求しました。

感情を煽るのではなく、判断を促すことを目的とし、あえて抑えたコピーと余白のある構成にしています。

働き方を変えることや、時間の使い方が人生を形作ることを伝える広告表現を目指しました。

Quietly

1日5分で始められるマインドフルネスアプリ「Quietly」の紹介用バナーです。

  • Graphic Design
  • Visual Design
概要
アプリ広告バナー
ツール / 言語
Photoshop / Illustrator
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

「忙しいあなたのための、5分リセット」をコンセプトに、短時間でも始められることが直感的に伝わるよう、情報量を抑えたシンプルな構成にしています。

アプリ画面を視線の中心に配置し、背景の流れが自然に視線を誘導するよう設計することで、サービス内容が一目で理解できるようにしました。

左下から右上へ流れるグラデーションと波のシェイプによって、「呼吸」「時間の流れ」「静かな朝」を視覚的に表現しています。

ユーザーの心理負担を減らし、自然に行動へつなげる設計を意識しました。

ストレスを感じやすい20代後半〜40代のビジネスパーソンをターゲットに、マインドフルネスアプリ「Quietly」の広告バナーを制作しました。

ネイビーを基調とした落ち着きのある配色に、パステルグリーンをアクセントとして加えることで、安心感の中に「始めやすさ」や「小さな変化」を感じられるよう設計しています。

アプリ画面を大きく配置することで、利用イメージを具体的に想起させ、ダウンロードへの心理的ハードルを下げることを意識しました。

短時間で内容が理解でき、自然に行動へつながる広告表現を目指しています。

チェキ風PC壁紙

チェキのような温かみのある質感を表現した、PC用オリジナル壁紙です。

  • Graphic Design
  • Visual Design
概要
デスクトップ壁紙
ツール / 言語
Photoshop
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

チェキのような温かみを表現するため、スクラッチや粒状感を加え、あえてダメージを残したレトロな質感に仕上げました。

彩度を抑えたやさしい色調と、少し色あせたトーンにすることで、なつかしさと落ち着いた空気感を演出しています。

写真の配置や余白のバランスを調整し、デスクトップ上でも情報の邪魔にならないよう、視認性と装飾性のバランスを意識しました。

長時間使用される壁紙として、視覚的な負担を抑えながら心地よさを保つ設計を意識しています。

チェキ写真を壁に飾ったような感覚で楽しめる、PC用オリジナル壁紙を制作しました。

単なる装飾ではなく、日常の作業画面に自然に溶け込むよう、主張しすぎないデザインを意識しています。

質感・色味・余白のバランスを整えることで、インテリアの一部のように使えるビジュアルを目指しました。

視覚的なノイズを抑えつつ、温かみと存在感を両立させた作品です。

Forest Friend

神秘的な光に包まれた森の広場で、静かに佇む鹿を描いたデジタルアート作品です。

  • Graphic Design
  • Visual Design
概要
デジタルアート
ツール / 言語
Photoshop
制作期間
約2時間（コンセプト設計・構図設計・ビジュアル制作）

工夫ポイント

森に迷い込んだときに出会う「ささやかな友」をテーマに、静かな安心感と余韻が同時に感じられる世界観を目指しました。

中央に配置した鹿に自然と視線が集まるよう、周囲の光やぼかしを調整し、奥行きのある空間表現を意識しています。

Adobe Fireflyで生成したイメージをベースに、油彩風のタッチやキャンバスのテクスチャを重ねることで、絵画のような温もりと質感を加えました。

神秘的な光に包まれた森の広場で、静かに佇む鹿を描いたデジタルアート作品です。

柔らかな光と影のコントラストによって、現実と幻想のあいだにあるような空気感を表現しました。

見る人がそれぞれの物語を重ねられるよう、情報を詰め込みすぎず、余白のある構成にしています。

日常の中でふと視線を向けたときに、心が少し静まるような体験を意識した作品です。
