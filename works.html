制作時間
2ヶ月
・ワイヤーフレーム…7日

これまでの経験や制作物をわかりやすく伝えるために、シンプルで見やすい構成を意識しながらポートフォリオサイトを制作しました。

このサイトを通して、私の雰囲気や大切にしている考え方などが少しでも伝われば幸いです。

LIMITSルーレット

  • Direction
  • Design
  • Coding
概要
自主制作｜ルーレット＋タイマーアプリ
ツール/言語
Visual Studio Code / HTML / CSS / Javascript
制作時間
2時間30分
・コンセプト…30分
・コーディング…2時間

デジタルアートバトル「LIMITS」の個人練習用に「テーマルーレット＋20分タイマーアプリ」を制作しました。
練習時に課題となりやすい「テーマに迷う時間」を解消し、LIMITSの緊張感や高揚感を、ひとつの画面で再現することを目指した練習用アプリケーションです。
具体（モノ）と抽象（テーマ）をルーレットで組み合わせ、両方確定と同時に公式ルールと同じ20分タイマーが自動スタートします。
北欧ナチュラル×犬をコンセプトに、集中を妨げない配色・余白設計と、START・TIME UP時に犬が跳び上がり、ぴょこんとおじぎするアニメーション演出を取り入れています。
LIMITS本番を想定した練習ツールとして、日々の制作トレーニングに活用しています。

1日5分で始められるマインドフルネスアプリ「Quietly」の紹介用バナーを制作しました。
