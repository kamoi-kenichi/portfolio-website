概要
訓練校課題｜架空の洋菓子店

架空のWebサイトを制作する課題で、HTML/CSS/JavaScriptを使った洋菓子店の店舗サイトを制作しました。
メインビジュアルの動画をはじめ、Slickスライドショーやメニューのふわっとしたアニメーションなど、JavaScriptを使った動きのある演出にも挑戦しています。

ポートフォリオ

  • Direction
  • Design
  • Copywriting
  • Coding
  • Retouch
概要
自主制作｜ポートフォリオ
ツール/言語
Visual Studio Code / HTML / CSS / jQuery / Illustrator / Photoshop / Canva
制作時間
2ヶ月
・ワイヤーフレーム…7日
・デザインカンプ…8日
・素材収集・制作…2週間
・コーディング…4週間

これまでの経験や制作物をわかりやすく伝えるために、シンプルで見やすい構成を意識しながらポートフォリオサイトを制作しました。
配色はライトベージュとチャコールグレーを基調に、アクセントカラーにモーブグレーを使用し、やわらかい雰囲気で落ち着いた配色の北欧ナチュラル感を演出しました。
細かい要素を足すことよりも、ミニマルな配色やシンプルで直感的な導線にこだわり、閲覧者の負担を減らしつつ、必要な情報に自然と辿り着ける構成にしています。
全体を通して、派手さよりも安心して見てもらえることを大切にしながら、作品としての完成度と自分らしさの両立を目指しました。
このサイトを通して、私の雰囲気や大切にしている考え方などが少しでも伝われば幸いです。

スピリットアニマル診断アプリ

  • Direction
  • Design
  • Copywriting
  • Coding
  • Retouch
概要
自主制作｜架空のPHPアプリ
ツール/言語
Visual Studio Code / HTML / CSS / jQuery / PHP / MAMP / Illustrator / Photoshop
制作時間
2日
・素材制作…1日
・コーディング…1日

自分の名前を入力して、文字コードをもとに「スピリットアニマル」を判定する診断アプリを制作しました。
名前を利用したシンプルなアルゴリズム設計で、ランダムではなく「名前に基づいた」診断結果を返す仕組みを実装しています。
ローディング画面やタイトルのエフェクト、グラデーションボタン等のアニメーションで「占いコンテンツらしいワクワク感」を演出し、印象に残るUI/UXを意識しながら制作しました。
訓練校で学んだPHPのフォーム処理や文字列操作、アニメーションを使った演出等の習熟度を図る目的で取り組みました。
今後は、複数の質問に答えるタイプの診断や、SNSシェア用のOGP画像生成などにもチャレンジしていきたいと考えています。

音楽ライブ

FALQON TOKYO

  • Design
  • Retouch
概要
自主制作｜架空ライブハウス
ツール
Illustrator / Photoshop
制作時間
2時間

渋谷区円山町の架空ライブハウス「FALQON TOKYO」
80〜90年代のレイヴカルチャーをベースに、近未来のテックビジュアルを融合させた音楽イベントのキービジュアルを制作しました。
ネオンカラーやグリッチ表現、立体感のあるタイポグラフィを取り入れることで、音に身を委ねる高揚感や、深夜のクラブ特有の没入感を表現しています。
クラシカルな彫刻像とヘッドフォンを組み合わせることで、時代を越えて受け継がれるグルーブを象徴的に描きました。
フライヤー、SNS告知の両媒体での使用を想定し、縮小表示でも印象が伝わるコントラストやレイアウトを意識しています。
レイヴ黎明期を知る世代から、現代のクラブシーンを楽しむ若年層まで、幅広い世代が同じフロアでサウンドを共有する“過去と未来をつなぐナイトイベント”がコンセプトのキービジュアルです。

映画バナー

Little Guardian

  • Design
  • Retouch
概要
自主制作｜ちいさな番人
ツール
Photoshop
制作時間
2時間

ハロウィンの夜にだけ開く、不思議な扉。
その前に佇むのは、かぼちゃの上に乗っている小さな番人。
かわいらしさの中に使命感を宿したキャラクターを中心に、ダークファンタジーと童話的要素を融合した世界観を描いています。
稲妻や満月、ゴーストといったハロウィンモチーフを取り入れながら、少しダークで、それでいてどこか温もりを感じさせるような、子どもから大人まで楽しめるハロウィンファンタジーを目指しました。
タイトルロゴは扉のアーチラインに沿わせることで、物語の入口を象徴し、作品の世界観と一体化する構成を意識しています。
夜の片隅でそっと世界を支える、小さな番人の静かな冒険を描いたキービジュアル作品です。

合成作品

星夢夜

  • Design
  • Retouch
概要
自主制作｜まどろみの向こう側
ツール
Illustrator / Photoshop
制作時間
2時間

就寝前、窓の外を見上げたその先に、広大な星空と神秘的な惑星が広がっていた――。
窓の向こうに広がる星空は、現実なのか、それとも夢の入り口なのか。
子供の空想の世界をテーマに、現実と夢の境界が曖昧になる瞬間をコラージュ作品で表現しました。
後ろ姿の少年を中心に構成することで、見る人が幼い頃の記憶や想像を重ねられる余白を残しています。
