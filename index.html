Game

Lumapuyo

JavaScriptで開発した、スマホ操作に対応したオリジナル落ちものパズルゲームです。 スワイプ操作による直感的なプレイと、連鎖による爽快感を重視し、 繰り返し遊びたくなる体験を目指して設計・実装しました。

  • Game Logic
  • JavaScript
  • Touch UI
  • Animation
  • Sound
概要
スマホ操作対応のオリジナル落ちものパズルゲーム
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript
制作期間
約2週間
（設計3日 / デザイン2日 / 素材2日 / 実装7日）

工夫したポイント

盤面管理や当たり判定、連鎖処理などのゲームロジックをJavaScriptで実装し、 状態管理を意識した設計を行いました。

また、スマホ操作でも快適に遊べるよう、タッチ操作の最適化にも工夫しています。

主な技術ポイント：
・スマホ操作を前提にしたスワイプ / タップ入力設計
・連鎖数に応じて変化する演出・効果音・スコア表示
・状態管理を分離したゲームループ構造
・requestAnimationFrameベースの描画制御
・UIレイヤー分離による演出制御設計

素材はすべてライセンス確認済み、または自作素材を使用しています。 企画・操作設計・演出・UI・音響まで一貫して設計・実装しました。 描画・演出・入力処理を分離し、保守性と拡張性を重視して設計しています。

App

LIMITSルーレット

JavaScriptで開発した、デジタルアートバトル「LIMITS」の個人練習向けテーマルーレット＋20分タイマーアプリです。

  • App
  • JavaScript
  • Animation
概要
具体（モノ）と抽象（テーマ）を組み合わせて出題し、20分タイマーを自動開始する練習アプリ
担当
企画 / UIデザイン / コーディング / アニメーション実装
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript
制作期間
約2日
（設計1日 / 実装1日）

課題と解決

練習時に課題になりやすい「テーマ決めに迷う時間」を減らし、発想から制作開始までをスムーズにできるよう設計しています。 また、テーマ確定から20分タイマー開始までを1画面で完結させることで、 本番に近い緊張感と高揚感を再現できる構成を目指しました。

具体（モノ）と抽象（テーマ）をルーレットで組み合わせ、両方の結果が確定すると、 公式ルールと同じ20分タイマーが自動でスタートする仕様にしています。

デザイン面では「北欧ナチュラル × 犬」をコンセプトに、集中を妨げない配色と余白設計を意識しました。

さらに、START時とTIME UP時には犬が跳び上がってぴょこんとおじぎするアニメーションと効果音を加え、 練習の緊張感の中にも親しみやすさを感じられる演出にしています。

ポートフォリオ

北欧ナチュラル × 犬をテーマに制作したポートフォリオサイトです。

  • HTML
  • CSS
  • JavaScript
  • UI/UX
概要
自身のスキルや制作物をわかりやすく伝えることを目的に、 情報設計・導線設計・ビジュアルデザインまで一貫して制作したポートフォリオサイト
ポイント
シンプルな構成により、初見ユーザーでも迷わず作品まで到達できる設計
担当
企画 / UIデザイン / コーディング / アニメーション実装
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript / Photoshop / Illustrator / Canva
制作期間
約2ヶ月
（設計2週間 / デザイン2週間 / 実装4週間）

工夫ポイント

ポートフォリオは「短時間で必要な情報にたどり着けること」が重要と考え、 情報量を絞りつつ、視線の流れに沿った導線設計を意識しました。 ミニマルな配色と余白設計により、見る人の負担を減らし、直感的に操作できるUIを目指しています。

「北欧ナチュラル × 犬」をコンセプトに、 やわらかく落ち着いた配色と余白を活かしたデザインにしています。

派手な演出ではなく、安心して閲覧できるトーンを重視し、 情報の読みやすさと世界観を両立させることを意識しました。

App

スピリットアニマル診断アプリ

PHPで開発した、自分の名前を入力して「スピリットアニマル」を診断するアプリです。

  • App
  • PHP
  • Animation
概要
名前をもとに診断結果を生成するスピリットアニマル診断アプリ（PHPによるフォーム処理・判定ロジック実装）
担当
企画 / UIデザイン / フロント実装 / PHPロジック実装
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript / PHP / MAMP / Photoshop / Illustrator
制作期間
約3日
（設計1日 / デザイン1日 / 実装1日）

工夫したポイント

占いコンテンツとしての「結果を見るまでのワクワク感」を重視し、 ローディング画面やタイトルエフェクト、グラデーションボタンのアニメーションを実装しました。
また、ランダムではなく「名前に基づいた結果」が返るようにすることで、 名前診断のように楽しめる体験を意識しています。

名前を利用したシンプルなアルゴリズム設計により、 同じ入力に対して同じ結果が返る仕組みを実装しています。

入力フォームから結果表示までの一連の処理をPHPで構築し、 フォーム処理や文字列操作などの基礎機能を活用しています。

ソライロ歯科医院

不安を感じやすい歯科診療において、「通いやすさ」と「相談しやすさ」を重視して設計した歯科医院サイトです。

  • HTML
  • CSS
  • JavaScript
  • UI/UX
概要
“通いやすさ” と “相談しやすさ”が特徴の歯科医院サイト（UI/UX設計）
担当
企画 / UIデザイン / コーディング / アニメーション実装
ツール / 言語
Visual Studio Code / HTML / CSS / JavaScript / Photoshop / Illustrator
制作期間
約2週間
（設計4日間 / デザイン3日間 / 実装1週間）

工夫ポイント

歯科医院に対して感じやすい「不安」や「緊張感」を軽減することを目的に、 空色を基調としたやわらかい配色と、丸みのあるUIデザインを採用しました。
また、アクセントとして温かみのあるオレンジを配置することで、 親しみやすさと視認性の両立を意識しています。

「通いやすさ」と「相談しやすさ」を軸に、 歯科医院に対する心理的なハードルを下げることを目的として制作しました。

初診予約や問い合わせへの導線を明確にし、 ユーザーが迷わず行動できる構成を意識しました。

UIデザインでは、丸みのあるレイアウトや円形の写真・装飾を取り入れ、 視覚的なやわらかさを演出しています。

また、ふんわりとしたアニメーションを加えることで、 落ち着いた印象と安心感のある体験を意識しました。

Design

FALQON TOKYO

架空ライブハウス「FALQON TOKYO」で開催される、 レイヴカルチャーと近未来を融合した音楽イベントのキービジュアルです。

  • Graphic Design
  • Visual Design
概要
80〜90年代のレイヴカルチャーと、近未来のテックビジュアルを融合させた音楽イベントのキービジュアル
ターゲット
クラブミュージックファン / 20〜30代 / ナイトイベント参加層
ツール / 言語
Photoshop / Illustrator
制作期間
約2時間

工夫ポイント

80〜90年代のレイヴカルチャーをベースに、 ネオンカラーやグリッチ表現、立体感のあるタイポグラフィを取り入れることで、 音に身を委ねる高揚感や没入感を視覚的に表現しました。

クラシカルな彫刻像とヘッドフォンを組み合わせることで、時代を越えて受け継がれるグルーブを象徴的に表現しました。

レイヴ黎明期を知る世代から、現代のクラブシーンを楽しむ若年層まで、幅広い世代が同じフロアでサウンドを共有する「過去と未来をつなぐナイトイベント」をコンセプトにしています。
